박예찬 KW Internet Portfolio

HTML, CSS, JavaScript로 만든 개인 포트폴리오이자 기말 프로젝트입니다.

실습 보기 대시보드 보기

Practice Pages

총 6개의 실습을 하나의 포트폴리오로 연결했습니다.

  • D-Day 계산기와 랜덤 뽑기
  • DOM 스타일 변경과 댓글 추가/삭제
  • 피자 주문서와 키보드 자동차

Extra Features

추가점수용 인터랙션을 별도로 구현했습니다.

  • 다크 모드, 스크롤 진행률, 타이핑 효과
  • 프로젝트 필터, 사이트 검색, 404 페이지
  • 방문 기록, 즐겨찾기, 메모, 체크리스트 저장

JavaScript

브라우저 저장소와 이벤트 처리를 적극 사용했습니다.

localStorage로 사용자 기록을 저장하고, click/input/submit/keydown 이벤트로 각 페이지가 직접 동작하게 만들었습니다.

Submission

GitHub Pages로 실제 웹 배포까지 완료했습니다.

로컬 서버가 꺼져도 제출 링크는 계속 유지되며, 없는 주소는 404 페이지로 안내합니다.
