# 박예찬 KW Internet Portfolio

인터넷활용 과제용 개인 포트폴리오 페이지입니다.

## 구성

- `index.html`: 메인 소개
- `about.html`: 자기소개와 제작 흐름
- `portfolio.html`: 프로젝트 카드와 필터
- `dday.html`: D-Day 계산 및 저장
- `random.html`: 랜덤 뽑기와 기록
- `contact.html`: 연락 폼, 입력 검증, 초안 저장

## 추가 기능

- 다크 모드 저장
- 스크롤 진행률 표시
- 타이핑 애니메이션
- 프로젝트 필터
- D-Day 일정 로컬 저장
- 랜덤 뽑기 기록 저장
- 연락 폼 글자 수 표시와 초안 저장
