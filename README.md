# 박예찬 KW Internet Portfolio

인터넷활용 과제용 개인 포트폴리오 페이지입니다.

자세한 기능 설명은 [`FEATURES.md`](FEATURES.md)에 정리했습니다.

## 구성

- `index.html`: 메인 소개
- `about.html`: 자기소개와 제작 흐름
- `portfolio.html`: 프로젝트 카드와 필터
- `dday.html`: 실습 1, D-Day 계산 및 저장
- `random.html`: 실습 2, 랜덤 뽑기와 기록
- `dom_style.html`: 실습 3, DOM 스타일 변환기
- `dom_list.html`: 실습 4, 댓글 추가/삭제
- `event_order.html`: 실습 5, 스마트 피자 주문서
- `event_car.html`: 실습 6, 키보드 자동차
- `dashboard.html`: 기말 프로젝트 대시보드
- `search.html`: 사이트 검색
- `presentation.html`: 발표 모드
- `404.html`: GitHub Pages 404 안내 페이지
- `contact.html`: 연락 폼, 입력 검증, 초안 저장

## 추가 기능

- 다크 모드 저장
- 스크롤 진행률 표시
- 타이핑 애니메이션
- 프로젝트 필터
- D-Day 일정 로컬 저장
- 랜덤 뽑기 기록 저장
- DOM 댓글 로컬 저장
- 피자 주문 영수증 표시
- 자동차 속도 조절과 화면 조작 버튼
- 실습 진행률 대시보드
- 실습 즐겨찾기와 최근 방문 기록
- 제출 체크리스트 저장
- 자기평가 퀴즈와 점수 저장
- 프로젝트 메모 자동 저장
- 사이트 검색과 키워드 칩
- 발표 모드 슬라이드와 방향키 조작
- GitHub Pages용 커스텀 404 페이지
- 실습 이전/다음 이동
- 대시보드 데이터 JSON 내보내기
- SEO/Open Graph 메타 태그
- 연락 폼 글자 수 표시와 초안 저장
